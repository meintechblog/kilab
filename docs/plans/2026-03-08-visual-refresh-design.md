# Visual Refresh Design

## Direction
- Adapted `ui-ux-pro-max` guidance: dark technical dashboard instead of a generic marketing landing page.
- Keep the current information architecture, but raise contrast, hierarchy, and visual intentionality.
- Use energy-market cues: graphite background, amber/orange highlight, green price context, red fixed-price reference.

## Visual System
- Background: deep graphite / midnight blue with layered radial glows.
- Typography: existing technical font stack kept, but with stronger mono accents and more compressed hierarchy.
- Surfaces: dark glass panels with brighter borders, stronger section separation, softer shadows.
- Data emphasis: KPI cards, breakdown panels, and chart become the primary visual anchors.

## Interaction
- Scenario cards stay the main control surface.
- Active scenario gets a stronger illuminated state.
- Chart legend and comparison lines stay readable on dark backgrounds.

## Constraints
- Preserve existing behavior and data flow.
- Keep mobile readability and keyboard-visible focus states.
- Avoid decorative churn that weakens scanability.
